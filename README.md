# 红岩网校寒假作业
## 后端部分除开gin以外使用了以下包：
1.viper，用于同步config文件；  
2.gorm，用于与mysql数据库交互;  
3.casbin，用于区分普通用户和管理员用户（代码里留了中间件但是并没有实现lol）;  
4.bcrypt，用于加密密码，避免明文储存，提高安全性；  
5.validator，用于进行入参校验（未实装XD，只在前端做了合法性校验）；  
6.JWT，用于生成token，进行用户校验；  
7.CORS，进行跨域配置（卡了两天）
## 前端部分使用vue框架：
1.aixos，与后端进行交互；  
2.ant-design-vue，方便地获取页面设计模板；
